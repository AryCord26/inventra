# 📦 Inventra

Inventra é um sistema moderno de **gestão de inventário e controle de estoque**, desenvolvido para simplificar o acompanhamento de produtos, entradas, saídas e relatórios em tempo real.

O objetivo do projeto é oferecer uma solução rápida, intuitiva e escalável para empresas que precisam de controle eficiente de estoque.

## ✨ Funcionalidades

- 📊 Dashboard com visão geral do estoque
- 📦 Cadastro, edição e remoção de produtos
- ➕ Controle de entrada de produtos
- ➖ Controle de saída de produtos
- 🔍 Busca e filtros inteligentes
- 📈 Relatórios de movimentação
- ⚠️ Alertas de estoque baixo (opcional)
- 👤 Controle de usuários (admin e operador)
- 📱 Interface responsiva (mobile e desktop)

## 🛠️ Tecnologias Utilizadas

- [Next.js](https://nextjs.org/) – Framework React fullstack
- [React](https://react.dev/) – Biblioteca de interface
- [TypeScript](https://www.typescriptlang.org/) – Tipagem estática
- [Tailwind CSS](https://tailwindcss.com/) – Estilização moderna
- [Axios](https://axios-http.com/) – Consumo de API
- [Node.js](https://nodejs.org/) – Backend (caso aplicável)
- [Prisma](https://www.prisma.io/) – ORM (caso esteja usando)
- [PostgreSQL](https://www.postgresql.org/) – Banco de dados

## 📁 Estrutura do Projeto

```bash
inventra-web/
├── app/                  # Rotas principais (Next.js App Router)
├── components/          # Componentes reutilizáveis
├── lib/                 # Utilitários e helpers
├── services/            # Integração com API
├── prisma/              # Schema do banco de dados
├── public/              # Arquivos estáticos
├── styles/              # Estilos globais
├── hooks/               # Custom hooks
├── context/             # Context API (estado global)
└── package.json
```
